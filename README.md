# Утилита для массовой установки ПО

### Установка
Скопируйте `install.sh` на нужное вам устройство, или возьмите `.deb` пакет из артефакта GitHub Actions.

### Использование
1. Создайте файл `install-config.txt` в рабочей директории скрипта. (Пример приложен в репозитории).
    В этом файле через перенос строки перечислите необходимое ПО для установки (по умолчанию ПО будет устанавливаться из пакетного менеджера)
2. Чтобы установить ПО, которого нет в пакетном менеджере по умолчанию, создайте скрипт `install-<пакет>.sh` в рабочей директории.
    Пример вы можете найти и прочитать в install-vitetris.sh. \
    Предоставляемые переменные:
    - `$PACKAGE_MANAGER` -- пакетный менеджер (`apt`/`dnf`/`pacman`)
    - `$UPDATE_COMMAND` -- команда для обновления репозиториев (например, `apt update`)
    - `$INSTALL_COMMAND` -- команда для установки, использующая пакетный менеджер (например, `apt install -y`)
3. Запустите скрипт (`sudo ./install.sh`)
    - Программа выведет вам план своих действий. Согласитесь, введя `y`.
    - Программа начнёт установку указанных в `install-config.txt` пакетов.
    - Программа выведет вам путь, где хранятся логи всех установок, а также общую сводку по результатам работы.
